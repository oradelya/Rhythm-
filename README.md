# 🎵 Rhythm — Платформа танцевальных воркшопов

Современное веб-приложение для поиска, просмотра и бронирования танцевальных воркшопов. Проект позволяет пользователям находить мероприятия, регистрироваться на занятия и взаимодействовать с организаторами.

---

# ✨ Возможности

- Регистрация и авторизация пользователей
- Просмотр танцевальных воркшопов
- Категории и фильтрация мероприятий
- Бронирование мест на воркшопы
- Личный кабинет пользователя
- Панель администратора
- Управление мероприятиями
- Адаптивный интерфейс
- Работа с базой данных через ORM

# ⚙️ Установка

```bash
# Клонировать репозиторий
git clone https://github.com/oradelya/Rhythm-.git

# Перейти в папку проекта
cd Rhythm-

# Создать виртуальное окружение
python -m venv venv

# Активировать venv
venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt

# Запуск проекта
python manage.py runserver
```
